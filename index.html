My Styles

This page contains specific style guide information for my front end interface.

Font

My application utilizes the Helvetica, Arial, "sans-serif" font-family with black text.

The Font-Awesome font-family is used for home icon and chevrons.

White text utilizes the fonts "Gill Sans", "Gill Sans MT", "Myriad Pro", "DejaVu Sans Condensed".

Header, Sub Header, Footer

The header and footer will be orange in color with white text.

The sub header is white with a solid gray border and black text.

Grid Component

An example of the grid component is used for page navigation.

The grid header or headers will be blue in color with a light gray border and black text.

Each row in the grid will alternate between a light gray color and white color.
